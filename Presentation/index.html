TypeBurn 🔥

A modern web app for testing and improving typing skills

What is the App?

An interactive platform for measuring typing speed and accuracy in multiple languages, with real-time feedback and leaderboards.

Problem Solved

  • Provides accessible typing practice
  • Gamifies skill improvement
  • Offers objective performance metrics
  • Enables progress tracking and competition

Key Features

  • 6 languages: EN, PL, DE, FR, ES, UK
  • Custom durations and difficulties
  • Live WPM and accuracy stats
  • Random nickname generator
  • Global leaderboard

User Experience

Pick a nickname, select language/duration/difficulty. Type the text with live feedback. View results and submit to leaderboard.

Backend Functionality

  • Session and data management
  • Language-specific word lists
  • Leaderboard storage
  • Anti-cheat validation
  • REST APIs

Technologies

Built with React for the frontend, Node.js and Express for the backend, Drizzle ORM for database management, and Vite for fast development. Uses TypeScript for type safety and Vitest for testing.

Architecture

Modular design: Client (UI), Server (logic/data), Shared (utilities). Client-server separation with common code sharing.

Future Improvements

  • More languages
  • User accounts
  • Better anti-cheat
  • Mobile support

Conclusion

The Typing Speed Test App offers a fun, effective way to improve typing skills through interactive tests and global competition.
